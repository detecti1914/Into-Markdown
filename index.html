Markdown 简介

什么是Markdown

Markdown是由著名科技博客Daring Fireball作者John Gruber发明的一种轻量标记语言，目的是为互联网内容发布者提供一个简单、易读/写，同时能够方便转换为HTML的书写工具。

The idea is that a Markdown-formatted document should be publishable as-is, as plain text, without looking like it’s been marked up with tags or formatting instructions.

--- John Gruber

Markdown的优点

  • 通用性：基于纯文本的轻量标记语言，不受平台、编辑器的限制，很多CMS内置对Markdown的支持
  • 易读 / 写性：文档可像普通文章一样阅读 / 书写，不像HTML一样有众多标记的干扰
  • 灵活性：
    • 可以像HTML那样为文字添加层级和样式
    • 可与HTML代码混合使用
    • 可方便的转换为HTML、PDF、RTF、Word，甚至是LaTeX
        ## Markdown实例
        
        基于纯文本的轻量[标记语言][1]
        [1]: http://daringfireball.com
        
        > Quote
        
        ### 优点
        
        * 通用
        * 易读 / 写
        * 灵活：
            * 可为文本添加层级、样式
            * 可与HTML代码混合使用
            * 方便的转换为多种格式

Mardown基础语法

文本样式：

			*斜体*    **粗体** 
			_斜体_    __粗体__

引用：

		> 引用使用'>'符号来标记
		>> 可嵌套。
		> 引用内部可以添加*文本样式*
  • 文字链接：
    • 行内链接： 
      格式：[Link Text](http://www.geekpark.net "title")
    • 引用链接： 
      格式：[Link Text][1]
					
					[1]: http://www.geekpark.net "Geekpark"
  • 图片链接：

    与文字链接类似，只是在标记的最开始加上’!’号

    ![Photo Alt](url "Photo Title" )

标题元素有两种写法：

风格一：

		H1
		========
		H2
		--------

风格二：

		# h1 #
		## h2 ##
		...
		###### h6 ######

列表：

顺序列表：

1. 第一项
		2. 第二项

无序列表：

* 第一项
		* 第二项

列表可嵌套，列表的每个项目可有多个段落

其他

		1. 两段间插入空行

		为两段添加<p>标签

		2. "`"符号为<code>标签

		一个Tab或四个空格为<pre>标签

		3. ---  ***  为水平分割线<hr />

		4. 在一行文字的最后打两个以上的空格符之后  
		回车换行将添加<br />标签

Markdown 扩展：

  • MultiMarkdown


    Markdown的一个扩展，为Markdown增加了文档内引用、脚注、链接样式、文档Meta信息、表格等功能。使其不仅仅可用于网页文档，还可以转换为LaTeX、PDF、RTF、Word文档，并保留其中的各种样式和元信息。
